# Graphy Weather

See your 5 day weather forecast in graphs


## Developers
### Install
```bash
cd .
npm install
```

Create a new file src/secrets.js
```javascript
export default {
  // get your key from from: https://home.openweathermap.org/api_keys
  API_KEY_OPEN_WEATHER_MAP: "MY_KEY_HERE"
};

```

```bash
npm start
```
